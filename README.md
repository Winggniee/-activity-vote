# 活动投票工具

一个简单的匿名投票工具，用于投票选择活动类型和参加时间。

## 功能特点

- ✅ 匿名投票，无需登录
- 🎯 支持多选活动类型
- 📅 自动生成接下来4个周末供选择
- 📊 实时查看投票结果
- 💾 数据本地存储（使用 localStorage）
- 📱 响应式设计，支持手机和电脑

## 活动选项

- 🔐 密室逃脱
- 🎭 剧本杀
- 🔫 真人CS
- 🎤 KTV
- 🏎️ 卡丁车
- 🎢 环球影城
- 🎊 Party Room

## 使用方法

1. 在浏览器中打开 `index.html` 文件
2. 选择你想参加的活动（可多选）
3. 选择你有空的周六日（可多选）
4. 点击"提交投票"按钮
5. 点击"查看结果"查看所有人的投票统计

## 技术栈

- HTML5
- CSS3
- JavaScript (原生)
- LocalStorage (数据存储)

## 部署

这是一个纯前端项目，可以通过以下方式部署：

1. **本地使用**：直接在浏览器中打开 `index.html`
2. **GitHub Pages**：上传到 GitHub 仓库并启用 GitHub Pages
3. **任何静态网站托管**：上传到 Netlify、Vercel、或任何支持静态网站的平台

## 让其他人参与投票

要让其他人也能参与投票，请查看 `DEPLOYMENT.md` 文件获取详细的部署指南。

简要步骤：
1. 在 [JSONbin.io](https://jsonbin.io/) 注册并创建 Bin
2. 将 Bin ID 和 API Key 配置到 `script.js`
3. 部署到 GitHub Pages、Netlify 或 Vercel
4. 分享链接给朋友们

## 注意事项

- 投票数据存储在云端（JSONbin.io），所有人共享同一个数据库
- 如果配置了后端，投票会自动同步给所有访问者
- 支持离线降级：如果网络失败会自动使用本地存储
- JSONbin.io 免费版每月有 10,000 次请求限制
